# HaPLa

**Source paper:** Joo, Koh, and Jung, *Harmful Prompt Laundering: Jailbreaking LLMs with Abductive Styles and Symbolic Encoding*, EMNLP 2025.

**Method:** Launders a request through abductive narrative styles and symbolic encodings to reduce direct harmful surface form.

I did not find a stable official code release, so no local prompt generator is
included.

**Paper link:** https://aclanthology.org/2025.emnlp-main.1296/

**Code link:** No stable official code repository found.

**Implementation:** Not migrated locally.

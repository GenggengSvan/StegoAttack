# AutoDAN

**Source paper:** Xiaogeng Liu et al., *AutoDAN: Generating Stealthy Jailbreak Prompts on Aligned Large Language Models*, ICLR 2024.

**Method:** Searches for stealthy jailbreak prefixes with an automated evolutionary procedure.

**Paper link:** https://openreview.net/forum?id=7Jwpw4qKkb

**Code link:** https://github.com/SheltonLiu-N/AutoDAN

**Implementation:** Not implemented locally because faithful AutoDAN requires the hierarchical genetic search loop and target-model feedback.

# GCG

**Source paper:** Andy Zou, Zifan Wang, Nicholas Carlini, Milad Nasr, J. Zico Kolter, Matt Fredrikson, *Universal and Transferable Adversarial Attacks on Aligned Language Models*, arXiv 2023.

**Method:** Optimizes an adversarial suffix with gradient-guided token search.

**Paper link:** https://arxiv.org/abs/2307.15043

**Code link:** https://github.com/llm-attacks/llm-attacks

**Implementation:** Not implemented locally because faithful GCG requires model logits, gradients, candidate sampling, and iterative suffix optimization.

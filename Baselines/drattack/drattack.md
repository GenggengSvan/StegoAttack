# DrAttack

**Source paper:** Xirui Li, Ruochen Wang, Minhao Cheng, Tianyi Zhou, Cho-Jui Hsieh, *Prompt Decomposition and Reconstruction Makes Powerful LLM Jailbreakers*, Findings of EMNLP 2024.

**Method:** Decomposes a request into less direct fragments and asks the model to reconstruct the intended task.

**Paper link:** https://aclanthology.org/2024.findings-emnlp.813/

**Code link:** https://github.com/xirui-li/DrAttack

**Implementation:** Not implemented locally because faithful DrAttack uses decomposition, in-context reconstruction examples, and synonym search.

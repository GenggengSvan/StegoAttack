# WordGame

**Source paper:** Zhang et al., *WordGame: Efficient & Effective LLM Jailbreak via Simultaneous Obfuscation in Query and Response*, Findings of NAACL 2025.

**Method:** Converts the request into a word puzzle so both the query and response are mediated through obfuscated words.

I did not find a stable official code release, so no local prompt generator is
included.

**Paper link:** https://aclanthology.org/2025.findings-naacl.269/

**Code link:** No stable official code repository found.

**Implementation:** Not migrated locally.

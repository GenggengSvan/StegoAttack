# Jailbroken

**Source paper:** Alexander Wei, Nika Haghtalab, Jacob Steinhardt, *Jailbroken: How Does LLM Safety Training Fail?*, NeurIPS 2023.

**Method:** Studies jailbreaks through two failure modes: competing objectives and mismatched generalization.

The paper is a taxonomy and analysis of jailbreak failure modes rather than a
single canonical attack template, so no local prompt generator is included.

**Paper link:** https://arxiv.org/abs/2307.02483

**Code link:** No stable official code repository found.

**Implementation:** Not migrated locally.
